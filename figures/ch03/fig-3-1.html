Richardson Maturity Model
Level 0: The Swamp of POX
Một URL duy nhất, POST everything
POST /api {action: 'getUser', id: 1}
Level 1: Resources
Nhiều URLs, mỗi URL = 1 resource
POST /users/1   POST /orders/5
Level 2: HTTP Verbs
GET/POST/PUT/DELETE đúng ngữ nghĩa
GET /users/1   DELETE /orders/5
Level 3: HATEOAS
Response chứa links điều hướng
{links: [{rel:"orders", href:"/users/1/orders"}]}
REST maturity
▲ cao hơn
Mức độ trưởng thành REST tăng dần từ Level 0 (RPC thuần túy) đến Level 3 (HATEOAS). Hầu hết các API thực tế đạt Level 2.
