🏫 LMS — Lộ trình đề xuất
Bước 1: Tạo VIEW cho Assignment đọc từ Core tables (không thay đổi code)
Bước 2: Tạo API endpoint trong Core để Assignment gọi thay vì query trực tiếp DB
Bước 3: Tách schema: core_schema và assignment_schema trong cùng PostgreSQL
Bước 4: Sync data qua Kafka CDC (Change Data Capture) từ core → assignment
Bước 5: Tách thành 2 PostgreSQL instances hoàn toàn độc lập
💡 Mỗi bước rollback-able. Không cần làm tất cả cùng lúc.
