🧩 JWT Structure (3 phần)
Header
{ "alg": "HS256", "typ": "JWT" }
Payload
{ "sub": "user123", "role": "STUDENT", "exp": 1712678400 }
Signature
HMACSHA256(header + "." + payload, secret)
🔍 Trong KBLab LMS: Auth Service generates JWT on login → client stores in localStorage → mọi request gửi kèm Authorization: Bearer <token> → Gateway (hoặc mỗi service) validates JWT bằng shared secret. Refresh token cho phép gia hạn mà không cần re-login.
