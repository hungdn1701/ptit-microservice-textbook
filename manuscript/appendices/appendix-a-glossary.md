# Phụ lục A: Thuật ngữ (Glossary)

<!-- TODO: Bổ sung đầy đủ thuật ngữ -->

| Thuật ngữ | Định nghĩa |
|---|---|
| **Aggregate** | Một cụm đối tượng domain được xem là một đơn vị dữ liệu nhất quán |
| **API Gateway** | Điểm truy cập duy nhất cho tất cả client requests đến hệ thống Microservices |
| **Bounded Context** | Ranh giới ngữ nghĩa mà trong đó một domain model có ý nghĩa nhất quán |
| **Circuit Breaker** | Pattern ngắt mạch khi một service downstream gặp sự cố |
| **CQRS** | Command Query Responsibility Segregation — Tách biệt mô hình đọc và ghi |
| **Event Sourcing** | Lưu trữ trạng thái dưới dạng chuỗi các sự kiện thay vì snapshot |
| **Eventual Consistency** | Mô hình nhất quán cuối cùng trong hệ thống phân tán |
| **Saga** | Pattern quản lý giao dịch phân tán bằng chuỗi local transactions |
| **Service Discovery** | Cơ chế để các service tìm và giao tiếp với nhau |
| **Strangler Fig** | Pattern di chuyển dần từ monolith sang microservices |
| **Ubiquitous Language** | Ngôn ngữ chung giữa domain experts và developers |
