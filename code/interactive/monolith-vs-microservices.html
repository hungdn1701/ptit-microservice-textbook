📊 Trade-off Comparison
Tiêu chí🏢 Monolith🌐 Microservices
Deploy speed✗ Toàn bộ app✓ Từng service
Scalability✗ Scale cả khối✓ Scale từng phần
Complexity✓ Đơn giản✗ Distributed systems
Data consistency✓ ACID tự nhiên✗ Saga/eventual consistency
Team autonomy✗ Coordination cao✓ Deploy độc lập
Debugging✓ Breakpoint trực tiếp✗ Distributed tracing
Tech stack✗ Duy nhất✓ Polyglot
