🔍 Trong KBLab LMS: Hệ thống sử dụng implicit saga (choreography via Kafka). Submit flow gồm 3-4 steps: Core Service tạo submission → Kafka → Judge Service chấm SQL → Kafka → Core Service cập nhật score → WebSocket notification. Khi Judge gặp lỗi, compensation = đổi status thành ERROR + thông báo "vui lòng thử lại".
