Defense in Depth — 4 Layers
Layer 1: Network
TLS, firewall
Layer 2: Gateway
JWT validation, rate limiting
Layer 3: Service
authorization, input validation
Layer 4: Data
encryption at rest, access control
Mỗi layer là một barrier. Attacker phải vượt qua TẤT CẢ layers.
LMS: TLS (HTTPS) -> Gateway JWT filter -> Service @PreAuthorize -> DB encryption
