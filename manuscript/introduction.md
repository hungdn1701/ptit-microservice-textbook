## Nguyên tắc tổ chức nội dung

Giáo trình tuân theo logic **"Nỗi đau → Giải pháp"**:

1. **Phân tách hệ thống** → Nảy sinh nhu cầu giao tiếp
2. **Cô lập dữ liệu** → Cần giao tiếp đồng bộ/bất đồng bộ
3. **Giao tiếp phức tạp** → Nảy sinh vấn đề nhất quán dữ liệu (Saga/CQRS)
4. **Nhiều dịch vụ** → Cần quản lý điểm truy cập (Gateway)
5. **Hệ thống phân tán** → Cần vận hành, bảo mật, quan sát
